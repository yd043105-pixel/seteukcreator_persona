# 학생부 세특 작성기 (세특 Creator · Persona)

학생부 종합전형(학종) 서류평가 기준에 맞춰 **세부능력 및 특기사항(세특)** 초안을 생성하는 단일 HTML 웹앱입니다.
설치·서버·로그인 없이 `세특작성기.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

> ⚠️ 생성 결과는 **초안**입니다. 반드시 교사가 검토·수정한 뒤 사용하세요.

## 주요 기능

### 멀티 AI 제공자
- **Anthropic Claude** (Opus 4.8 / Sonnet 5 / Haiku 4.5)
- **Google Gemini** (3.1 Pro / 3.5 Flash / 3.1 Flash-Lite / 2.5 Pro)
- 모델 ID 직접 입력 지원
- API 키는 서버를 거치지 않고 브라우저에서 각 제공자 API로 **직접 전송**됩니다.

### 두 가지 생성 모드
| 모드 | 호출 수 | 설명 |
|---|---|---|
| ⚡ 빠른 생성 | 1회 | 단일 호출로 즉시 초안 생성 |
| 🏆 정밀 생성 | 10회 | 2라운드 평가위원회를 거쳐 완성도를 끌어올림 |

**정밀 생성 파이프라인 (2라운드)**

```
1단계  초안 작성 (베테랑 교사)
2단계  1R 평가 — 학업역량 · 진로역량 · 공동체역량 (3인)
3단계  종합평가위원 개선본 작성
4단계  2R 재평가 — 위 3인 + 기재요령·문체 전문가 (4인)
5단계  종합평가위원 최종본 작성
```

2라운드 평가자에게는 **자신이 1차에서 지적한 사항**이 전달되어, 반영 여부를 확인하고 점수를 상향합니다.
평가자는 `score` / `strengths` / `improvements` / `to_perfect`(100점 기준)를 JSON으로 반환합니다.

### 평가 기준
주요 대학 공통 평가요소를 프롬프트에 내장했습니다.
- **학업역량 (50%)** — 학업성취도 · 학업태도 · 탐구력
- **진로역량 (30%)** — 전공 관련 교과 이수 노력 · 성취도 · 진로 탐색 활동
- **공동체역량 (20%)** — 협업과 소통 · 나눔과 배려 · 성실성과 규칙준수 · 리더십

### 기재 금지 규칙 (자동 적용)
- 수상 실적, 모의고사·석차, 교외 활동, 부모 정보, 특정 대학·기관명, 학생 실명 등
- **생성형 AI·챗봇 활용 언급 금지** (학생이 실제 사용했더라도 미기재)
- **교육과정 범위 초과·선행학습 암시 표현 금지** → "교육과정에서 배운 개념을 확장·연결하여"로 서술

### 입력
- **교과**: 과목명 / **비교과**: 자율·진로·동아리 + 세부 활동
- 학생의 정성적 특성(수업에서 드러난 학습 태도·강점)
- 참고 자료: PDF · 이미지 · 텍스트 파일 첨부(여러 개) 또는 직접 붙여넣기
- 목표 글자 수 (영역별 자동 설정: 자율·동아리 500자, 진로 700자)

### 일괄(배치) 생성
- 학생 카드를 **원하는 만큼 추가**하고 한 번에 순차 처리 (기본 1명)
- 학생별 독립 파일 첨부 / 진행표(상태·점수·글자수) / 중단 / 실패·미처리만 재시도
- 결과 **txt · CSV · JSON** 내려받기, localStorage 자동 저장·복원
- 학생 식별자(이름·학번)는 결과 정리용일 뿐 **AI 프롬프트에 전달되지 않습니다**

### 무료 한도 대량 처리 (Gemini)
- **다중 API 키**: 한 줄에 하나씩 입력하면 라운드로빈으로 분산 → 한도가 키 개수만큼 배수
- **레이트 리미터**: RPM · TPM 자동 준수, 429 지수 백오프, 일일 한도 소진 시 키 교체
- **학생 간 페이싱**: `호출수 ÷ (RPM × 키 개수) × 60초` 자동 계산
  - 예) 정밀 · RPM 10 · 키 1개 → **60초 (1분당 1명)** / 키 2개 → 30초
- 한도 안내 박스에 "하루 약 N명 처리 가능" 실시간 표시

## 사용법
1. `세특작성기.html`을 브라우저로 엽니다.
2. AI 제공자를 고르고 API 키를 입력합니다.
   - Anthropic: https://console.anthropic.com/settings/keys
   - Google AI Studio: https://aistudio.google.com/app/apikey
3. 교과/비교과와 과목(또는 영역)을 선택합니다.
4. 학생의 정성적 특성을 적고, 산출물을 첨부합니다.
5. **빠른 생성** 또는 **정밀 생성**을 실행합니다.

### 대량(수십~120명) 작업 팁
1. Gemini 키를 여러 개 발급해 한 줄에 하나씩 입력
2. 모델은 **Flash / Flash-Lite** 선택 (일일 한도가 Pro보다 큼)
3. **일괄 빠른 생성**으로 전체 초안 확보 → CSV 내려받기
4. 중요한 소수 학생만 **정밀 생성**으로 완성도 극대화

> 장시간 배치는 브라우저 탭을 **화면에 띄워둔 채** 실행하세요. 백그라운드 탭은 타이머가 느려집니다.

## 개인정보
- 서버가 없으며 파일·키·학생 정보가 외부로 전송되지 않습니다(선택한 AI 제공자 API 제외).
- API 키는 선택 시에만 브라우저 localStorage에 저장됩니다. **공용 PC에서는 저장하지 마세요.**
